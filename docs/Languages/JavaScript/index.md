## Exercises

### Beginner exercises

- identify where JavaScript runs in the browser
- explain the difference between `var`, `let`, and `const`

### Intermediate exercises

- write a small closure example
- explain why `===` is preferred over `==`

### Advanced exercises

- explain the event loop in your own words
- compare object references with primitive values
